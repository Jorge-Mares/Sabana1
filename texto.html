Las etiquetas de texto en HTML son aquellas que permiten introducir texto y modificar su formato de texto en html, de manera que se adapte a los deseos e intereses de cada desarrollador web. En este sentido, están fuertemente vinculadas con el lenguaje de estilo CSS. Ahora sí, a continuación te mostramos algunas de las etiquetas de texto en HTML más usadas: <"em">: el elemento <"em"> se usa para enfatizar algunas partes del documento HTML. Es decir, nos permite identificar algunas frases u contenido, de manera que no sean diferentes solo visualmente, sino que también se puede identificar su importancia internamente o desde el código. <"strong">: esta etiqueta para escribir texto en html es similar al elemento <"em">, solo que proporciona mayor importancia. Es decir, usamos <"strong"> para hacer mayor énfasis que el permitido por la etiqueta <"em">. Esta etiqueta también está vinculada a las etiquetas semánticas de HTML, ya que dota de mayor sentido al texto y permite una estructuración más clara e identificable. Por esta razón, la etiqueta <"strong"> está fuertemente relacionada con Search Engine Optimization. <"small">: la función de la etiqueta <"small"> es reducir el tamaño de la letra al mínimo. Esto resultad de mucha utilidad a la hora de integrar comentarios, cuadros de diálogos, referencias a autores o fuentes y otra información que, según el diseño, necesitemos ubicar al margen y a menor escala. <"mark">: esta es una de las etiquetas de texto en HTML que permite resaltar de color amarillo algunas partes del texto de especial relevancia. Aunque puede confundirse con la etiqueta strong, esta última identifica la importancia, mientras que la etiqueta mark resalta la relevancia de algunos intervalos de texto. La relevancia es una posibilidad, es decir, información que pueda ser relevante para unos usuarios, pero que no es de importancia imprescindible al texto. <"s">: la etiqueta <"s"> es una etiqueta con apertura y cierre que nos permite tachar el texto con una línea horizontal a la mitad así. <"i">: en cuanto a la etiqueta <"i">, esta es la encargada de pintar el texto en cursiva o letra inclinada. Cabe resaltar que también es de uso obligatorio el inicio y cierre y que no posee atributos específicos, solo atributos genéricos como title, style o id. <"b">: la etiqueta <"b"> por su parte, le da formato en negrita al texto y su definición precisamente es bold, que se traduce como negrita. Sin embargo, es importante que recuerdes que algunos desarrolladores recomiendan usar el lenguaje CSS para este tipo de modificaciones. <"br">: por otra parte, tenemos la etiqueta <"br"> de line break, que permite definir saltos de línea, sobre todo en aquellos documentos en los que las divisiones de línea son importantes, como al momento de introducir poesía, canciones o direcciones. <"u">: por último, otra de las etiquetas de texto comunes es la etiqueta <"u">, que es la encargada de presentar el texto subrayado. Esta etiqueta también requiere obligatoriamente la etiqueta de apertura y cierre. ¿Cómo aprender más sobre etiquetas de texto en HTML?
